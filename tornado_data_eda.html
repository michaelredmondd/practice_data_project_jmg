The Data

We obtained the raw data from [here(https://www.spc.noaa.gov/wcm/data/1950-2022_actual_tornadoes.csv)]. It contains observations about tornado events from 1950-2022. The data is downloaded in the R script “get_tornados_data.R”. The cleaned data is saved in the data folder as “tornados.csv”

Let’s load the data:

Build our map data and filter out AK, HI, PR, and VI

tornados_49 <- tornados %>%
  filter(st != "AK", st != "HI",st!= "PR", st!= "VI")
tornados_49 %>%
  ggplot(aes(x = st)) + 
  geom_bar() + 
  coord_flip()

From which we see the states with the most tornados:

tornados_49 %>%
  group_by(st) %>%
  summarise(per_state = n()) %>%
  arrange(desc(per_state)) %>%
  ggplot(aes(x=reorder(st,per_state),y=per_state)) + 
  geom_bar(stat = "identity") + 
  coord_flip()

Filter out (that is, remove) those observations where the ending latitude or longitude is 0.

tornados_49 <- tornados_49 %>%
  filter(elat > 0 & elon < 0)
tornados_49 %>%
  ggplot() + 
  geom_polygon(data=states_df,
               mapping = aes(long,lat,group=group),
               fill="white",color="darkgray") + 
  geom_segment(aes(x=slon,y=slat,xend=elon,yend=elat), alpha=0.2)

Remove observations from the West.

tornados_49 <- tornados_49 %>%
  filter(slon >= -105)

states_df <- states_df %>%
  filter(long >= -105)
tornados_49 %>%
  ggplot() + 
  geom_polygon(data=states_df,
               mapping = aes(long,lat,group=group),
               fill="white",color="darkgray") + 
  geom_segment(aes(x=slon,y=slat,xend=elon,yend=elat),alpha=0.5,color="darkgreen")
